The `MyList` class from `1-my_list` module
==========================================

Using `MyList`
--------------

Import the class:

    >>> MyList = __import__('1-my_list').MyList

Adding elements and printing sorted:

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Edge cases:

    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

    >>> my_list = MyList([10])
    >>> my_list.print_sorted()
    [10]

    >>> my_list = MyList([5, 3, 9, 1])
    >>> my_list.print_sorted()
    [1, 3, 5, 9]
